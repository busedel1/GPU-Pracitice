2nd order Wave Equation Demo

このデモは２次の波動方程式を CPU または GPU 上で計算します。
このデモは「Parallel Computing Toolbox」およびサポート対象 GPU カードを要求いたします

GUI:
* "WaveEqnGUI.m" を実行 
* "CPU" または "GPU" を選択
* グリッド・サイズの入力。サイズが大きいほど計算時間を要します。
* "Start" をクリック
* 上のグラフは波動方程式の解を表し、下のグラフは１期前との差分を表します。
* WaveEqn_CPU.m と WaveEqn_GPU.m とのアルゴリズムを比較してみましょう。

Script:
* "WaveScript.m" を開いて１ステップごとにセルモード実行してみましょう。
